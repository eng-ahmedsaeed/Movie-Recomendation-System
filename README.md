# Movie Recommendation System 

A Java-based application developed for a Software Testing course. This system processes user preferences and movie datasets to generate personalized, genre-based suggestions. The project emphasizes robust software quality through various testing methodologies.

## Features
* **Data Processing:** Parses movie and user data from text files.
* **Validation:** Robust input validation for user data and movie metadata.
* **Recommendation Engine:** Generates movie suggestions based on genre affinity.
* **Comprehensive Testing:** Implementation of multiple testing levels to ensure reliability.

## Tech Stack
* **Language:** Java
* **Build Tool:** Maven (using `pom.xml`)
* **Testing Framework:** JUnit

##  Testing Scope
This project serves as a practical implementation of various testing strategies:
* **Unit Testing:** Validating individual methods and logic.
* **Integration Testing:** Ensuring different modules work together correctly.
* **White-Box Testing:** Path and logic coverage analysis.
* **Black-Box Testing:** Testing requirements and functionality without internal code knowledge.
